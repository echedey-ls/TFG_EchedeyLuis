# Trabajo Fin de Grado

En este repositorio acojo los documentos fuentes de mi TFG.

Está basado en la plantilla que ofrece la ETSIINF en Overleaf.

## Cómo reutilizar este repositorio

Varias notas:

0. Se ha desarrollado con VSCode exclusivamente y la extensión de LaTeX Workshop.
1. Lee el `README_ETSIINF.md` para entender cómo funciona la plantilla.
2. Hay múltiples cambios que se han realizado, algunas adiciones de comandos para adaptar la plantilla a mis necesidades.
3. Se añade un glosario, el jurado siempre lo agradece.
4. No tomes mi TFG como muestra de un buen TFG, porque no lo es. Es un TFG, y ya está:
  - Se recomienda aumentar la extensión de algunas secciones.
  - Se recomienda explicar más detalladamente algunos procedimientos, como el proceso de cómo contribuir.
